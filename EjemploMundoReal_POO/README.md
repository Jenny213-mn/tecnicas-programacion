
## Ejemplos del Mundo Real – Programación Orientada a Objetos

En la carpeta `EjemplosMundoReal_POO` se desarrollaron tres ejemplos del mundo real:
- Sistema de reservas de hotel
- Sistema de tienda
- Sistema de biblioteca

Cada ejemplo aplica los principios de la Programación Orientada a Objetos mediante el uso de clases, atributos y métodos.
