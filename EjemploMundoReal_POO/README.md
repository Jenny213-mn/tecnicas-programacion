## Ejemplos del Mundo Real – Programación Orientada a Objetos (POO)

Este repositorio contiene diferentes actividades y ejercicios desarrollados durante la asignatura **POO**, con el objetivo de aplicar conceptos fundamentales de programación y buenas prácticas de desarrollo.

En particular, en la carpeta **`EjemplosMundoReal_POO`** se presentan varios programas escritos en **Python** que modelan situaciones del mundo real utilizando los principios de la **Programación Orientada a Objetos (POO)**.


## EjemplosMundoReal_POO

Dentro de esta carpeta se incluyen los siguientes ejemplos:

### Sistema de Reservas de Hotel (`sistema_reservas.py`)

Este programa representa un sistema básico de reservas de un hotel. Se utilizan clases para modelar entidades reales como el hotel y las habitaciones. Cada habitación posee atributos como número, tipo, precio y disponibilidad, y métodos que permiten gestionar su reserva.

Este ejemplo demuestra cómo la POO facilita la organización del código y la interacción entre objetos.

### Sistema de Tienda (`tienda.py`)

Este ejemplo modela el funcionamiento de una tienda sencilla. Se utilizan clases para representar productos y clientes, permitiendo simular la compra de artículos y el cálculo del total a pagar.

El programa evidencia el uso de atributos, métodos y la relación entre objetos para resolver un problema del mundo real.

### Sistema de Biblioteca (`biblioteca.py`)

En este programa se simula el funcionamiento de una biblioteca. Se emplean clases para representar libros y usuarios, permitiendo prestar y devolver libros de manera controlada.

Este ejemplo refuerza el uso de la POO para manejar estados de los objetos y sus interacciones.
